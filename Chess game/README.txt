The main.cpp contains the code , the graphics library used in this project is called "Raylib"
the executable is "game.exe"
Press R to Restart the game
Press M to show the Pause Menu
LMB (Left Mouse Button) : Select a Chess piece 
RMB (Right Mouse Button) : Move the Chess piece (LMB can do the same action too)
to stop the background music , go to Option and press the mute icon (you can find it on the Pause menu aswell)
